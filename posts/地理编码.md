地理编码是将 SAR 雷达坐标系的数据转为地理坐标系的步骤，这涉及到两个概念：

- **SAR 雷达坐标系**：以雷达斜距向和方位向为基准的坐标系
- **地理坐标系**：以经纬度为基准的坐标系

> 本文涉及的所有指令均已收录至指令速查库，点击高亮指令名即可查看完整帮助文档和参数翻译。

## 0. 导入 DEM

如果 DEM 是自己裁剪的范围，一般是 TIFF 格式，需要在 GAMMA 中使用 :cmd[dem_import] 指令。

**dem_import**

Convert original DEM file format into Gamma format, transform geoid heights into ellipsoid heights.

将原始 DEM 文件转换为 GAMMA 格式，并将大地水准面高转换为椭球高。

> **注意**：大地水准面用于将 DEM 高度从海拔高（相对于大地水准面）转换为椭球高。GAMMA 自带 EGM96 和 EGM2008 全球水准面文件，位于 `$DIFF_HOME/scripts/` 目录下。

常用调用示例：

```bash
dem_import demo.dem.tif demo.dem demo.dem.par 0 1
```

含义：输入 GeoTIFF 格式的 `demo.dem.tif`，输出 Gamma 格式的 `demo.dem` 和 `demo.dem.par`，优先使用输入文件自身的投影参数。

## 1. 生成多视强度图

对 SLC 影像进行多视处理，生成多视强度图（MLI），提高信噪比并使距离向和方位向的像素间距接近一致。使用 :cmd[multi_look] 指令。

## 2. 粗查找表与模拟 SAR 生成

根据 DEM 的空间分辨率和 SAR 影像的像素间距计算过采样因子，结合精确轨道数据和 DEM，生成初始的粗查找表（Rough Lookup Table），并将 DEM 转换生成雷达坐标系下的模拟 SAR 强度图。

流程：:cmd[gc_map] → :cmd[geocode]

**gc_map**

Calculate terrain-geocoding lookup table and DEM derived data products.

计算地形地理编码查找表及 DEM 衍生数据产品（模拟 SAR 强度图、局部入射角、叠掩阴影图等）。

**geocode**

Forward geocoding transformation using a lookup table.

前向地理编码转换，使用查找表将雷达坐标系数据重采样到地理坐标系。

## 3. 精配准

创建偏移量参数文件，首先进行全局宏观偏移的初始估计。随后对真实多视强度图（MLI）与模拟 SAR 图像执行多视互相关匹配，并剔除误差点，计算出两者之间的精确映射关系。

流程：:cmd[create_diff_par] → :cmd[init_offsetm] → :cmd[offset_pwrm] → :cmd[offset_fitm]

**create_diff_par**

Create DIFF/GEO parameter file for geocoding and differential interferometry.

创建 DIFF/GEO 参数文件，用于地理编码和差分干涉测量。

**init_offsetm**

Initial offset estimation for multi-look intensity images.

多视强度图初始偏移量估计，通过互相关确定参考影像与待配准影像之间的大致偏移。

**offset_pwrm**

Offset estimation between MLI images using intensity cross-correlation.

基于强度互相关的多视影像偏移量精估计，在多个窗口上计算距离向和方位向的亚像元级偏移。

**offset_fitm**

Range and azimuth offset polynomial estimation using SVD.

使用 SVD 方法拟合偏移多项式，剔除误差点后建立精配准映射关系。

## 4. 生成精查找表与坐标系转换

将上一步拟合好的偏移量参数应用到粗查找表中，生成用于最终转换的精细查找表（fine lookup table）。利用该表，将地理坐标系下的 DEM 精确重采样到雷达坐标系下。

流程：:cmd[gc_map_fine] → :cmd[geocode]

**gc_map_fine**

Geocoding lookup table refinement using DIFF_par offset polynomials.

利用偏移多项式对粗查找表进行精化，生成可用于精确地理编码的精细查找表。

## 5. 向后地理编码

把雷达坐标系下的影像转为地理坐标系下的影像。使用 :cmd[geocode_back] 指令。

**geocode_back**

Geocoding of image data using a geocoding lookup table.

向后地理编码，使用查找表将雷达坐标系影像转换到地理坐标系（与 geocode 方向相反：geocode 是雷达→地理，geocode_back 是地理→雷达）。
